# Comment Statistics UI (CityFeedback)

Интерфейс админ-панели для проекта мониторинга и классификации комментариев.

## Что реализовано

- JWT-авторизация (мок): страница логина `admin / admin123`
- Раздел `Настройки`:
  - источники VK (добавление/удаление)
  - email получатели отчетов (добавление/удаление)
  - период запуска парсера в минутах
  - блок лицензии: статус, срок, лимиты, ввод ключа
- Раздел `Статистика`:
  - выбор диапазона дат
  - формирование отчета
  - таблица по 9 категориям
  - общее количество комментариев
  - скачивание отчета (.xlsx)
  - отправка отчета на email (мок)

## Быстрый старт

```bash
cd comment-statistics-ui
npm install
npm run dev
```

## Технические заметки

- API пока на моках в `src/api/mockApi.ts`.
- Для прод-версии frontend должен ходить только в backend.
- В production лицензионный ключ не хранится во frontend.
